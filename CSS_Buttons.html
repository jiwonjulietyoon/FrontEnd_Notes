Responsive Button Designs

General Tips on Creating Custom Buttons with Pure CSS

  1. Create a button container, i.e. an extra <div> tag encompassing the actual button component, especially if there are multiple buttons.
    An outer container makes it easier to control button position or any transformation effects.
  2. Use paddings or width/height properties to control button size:
    • Specific width & height: Button size will remain consistent regardless of content (button label). Need to take note of content length, since long content may overflow
    • Padding: Button size will stretch to fit its content. Adding the "white-space: nowrap;" property will prevent the button from breaking up into multiple lines when viewport width decreases.
  3. <div> tags v.s. <button> tags for button component:
    • Button: Semantic tag => easier to distinguish among a number of div tags. Setting the button type (button|submit|reset) is also possible for when it is used in forms.
    • Add "cursor: pointer;" to indicate that the button element is clickable
    • For <button>, add "border: none;" and "button:focus {outline: none;}" to get rid of the border and outline, respectively.
  4. Create 3-4 modes: default (no user interactions), hover (cursor hovers over the button), active (the moment when the button is clicked), and disabled (inactive button)

Nudge Buttons

HTML

<button class="nudge-btn nudge-primary">Primary</button>
<button class="nudge-btn nudge-secondary">Secondary</button>
<button class="nudge-btn nudge-disabled" disabled>Disabled</button>

Common Properties

.nudge-btn {
  border: none;
  cursor: pointer;
  display: inline-block;
  position: relative;
  padding: 10px 20px;
  border-radius: 3px;
  color: white;
  text-transform: uppercase;
  white-space: nowrap;
}
.nudge-btn:focus {
  outline: none;
}

Primary

.nudge-primary {
  background: #00C0FF;
  box-shadow: 5px 0 #00A2FF;
}
.nudge-primary:hover {
  box-shadow: 3px 0 #00A2FF;
  left: 2px;
}
.nudge-primary:active {
  box-shadow: none;
  left: 5px;
}

Secondary

.nudge-secondary {
  background: #9AAEBB;
  box-shadow: -5px 0 #8D9CB2;
}
.nudge-secondary:hover {
  box-shadow: -3px 0 #8D9CB2;
  right: 2px;
}
.nudge-secondary:active {
  box-shadow: none;
  right: 5px;
}

Disabled

.nudge-disabled {
  background: #d6dee3;
  box-shadow: 5px 0 #ccd6dd;
  cursor: initial;
}

Slant Button

Slant Buttonarrow_forward

HTML

<div class="slant-btn">
  slant Button
  <i class="material-icons">arrow_forward</i>
</div>

CSS

.slant-btn {
  cursor: pointer;
  overflow: hidden;
  display: inline-block;
  position: relative;
  padding: 10px 15px 17px 25px;
  background: #00A2FF;
  color: white;
  text-transform: uppercase;
  font-weight: bold;
  white-space: nowrap;
  transition: all 0.3s;
}
.slant-btn i {
  transform: translateY(6px);
  margin-left: 50px;
}
.slant-btn::before {
  content: "";
  width: 70px;
  height: 100px;
  position: absolute;
  right: -20px;
  top: -45%;
  background: #00C0FF;
  transform: rotate(-20deg);
  transition: all 0.3s;
}
.slant-btn:hover::before {
  width: 90px;
}
.slant-btn:active {
  transform: translateY(2px);
}

Slide Button

Slide Button

HTML

<div class="slide-btn">Slide Button</div>

CSS

.slide-btn {
  cursor: pointer;
  display: inline-block;
  padding: 5px 15px;
  border: 2px solid #00A2FF;
  background-image: linear-gradient(to right, #00A2FF 50%, transparent 50%);
  background-size: 200% 100%;
  background-position: right;
  color: #00A2FF;
  white-space: nowrap;
  transition: all 0.3s;
}
.slide-btn:hover {
  background-position: left;
  color: white;
}

Glow Button

Double Border Button

Double Border

HTML

<div class="dbl-border-btn">Double Border</div>

CSS

.dbl-border-btn {
  cursor: pointer;
  display: inline-block;
  position: relative;
  padding: 10px 20px;
  background: rgba(0, 0, 0, 0.1);
  border: 1.5px solid #00A2FF;
  color: #00A2FF;
  white-space: nowrap;
  transition: all 0.3s;
}
.dbl-border-btn::before, .dbl-border-btn::after {
  content: "";
  box-sizing: border-box;
  width: 95%;
  height: 135%;
  border: 0px solid #00A2FF;
  transition: all 0.3s;
  position: absolute;
  top: 50%;
  left: 50%;
}
.dbl-border-btn::before {
  border-top-width: 1.5px;
  border-bottom-width: 1.5px;
  transform: translate(-50%, -50%) scaleX(0);
  transform-origin: left;
}
.dbl-border-btn::after {
  border-left-width: 1.5px;
  border-right-width: 1.5px;
  transform: translate(-50%, -50%) scaleY(0);
  transform-origin: top;
}
.dbl-border-btn:hover {
  background-color: rgba(0, 0, 0, 0);
}
.dbl-border-btn:hover::before {
  transform: translate(-50%, -50%) scaleX(1);
}
.dbl-border-btn:hover::after {
  transform: translate(-50%, -50%) scaleY(1);
}
